# SAC-Skill-Tracker

SAC Skill Tracker is a web app that helps the Shippensburg Aquatic Club track the progress of their swimmers’ skill development. The platform allows coaches to record, update, and monitor individual skill milestones, making it easier to understand each swimmer’s growth over time.

The application is built with **Next.js on the frontend** to provide a fast, responsive, and modern user experience. The **backend is powered by Supabase**, which handles the database, authentication, and real-time data updates. This stack enables a simple, scalable, and efficient system for managing swimmer records and skill progress.

Key features include:
- Centralized tracking of swimmer skills and milestones
- Easy updates by coaches during or after practice
- Real-time data storage and retrieval through Supabase
